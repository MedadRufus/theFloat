


Get a view of the size of symbols with first installing `binutils-avr`
```bash
sudo apt install binutils-avr
```
Then build the project with platform.io. We shall then inspect the executable for largest functions with:

```bash
avr-nm .pio/build/pro8MHzatmega328/firmware.elf --print-size --size-sort
```
